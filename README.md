# Leads-tracker
A simple and lightweight Chrome extension that helps you save your open tabs for later use. Tabs are stored safely in localStorage, so you can easily restore them even after closing your browser.

# Features:

Save all or selected tabs with one click.

Automatically store tab data using localStorage.

Restore saved tabs anytime.

Option to clear saved sessions.

Fast, minimal, and easy to use.

# Tech Stack:

HTML, CSS, JavaScript.

Chrome Extension API.

LocalStorage for data persistence.
